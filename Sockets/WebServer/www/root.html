You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
    Example: http://98.82.7.191:9000/file/sample.html
  • /json -- returns a JSON response for a random image
    Example: http://98.82.7.191:9000/json
  • /random -- returns index.html
    Example: http://98.82.7.191:9000/random
  • /multiply?num1=3&num2=4 -- multiplies two numbers and returns the result
    Example: http://98.82.7.191:9000/multiply?num1=3&num2=4
  • /github?query=users/amehlhase316/repos -- retrieves/displays details of the specified GitHub repositories
    Example: http://98.82.7.191:9000/github?query=users/amehlhase316/repos
  • /calculateAge?birthYear=1990¤tYear=2024 -- calculates your age based on provided birth and current years
    Example: http://98.82.7.191:9000/calculateAge?birthYear=1990¤tYear=2024
  • /calculateDistance?x1=1&y1=1&x2=4&y2=5 -- calculates the distance between two points on a plane
    Example: http://98.82.7.191:9000/calculateDistance?x1=1&y1=1&x2=4&y2=5
